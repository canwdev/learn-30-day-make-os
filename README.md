# learn-30-day-make-os

《30天自制操作系统》学习代码

## 备注

- 需要在 Windows 环境下运行。
- 请从「[30天自制操作系统光盘](https://gitee.com/paud/30daysOS/tree/master/tolset/z_tools)」复制 `z_tools` 工具文件夹，并放置在此目录方可正常运行。
